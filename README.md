# Модуль 02 – Базовые принципы проектирования

## Используемые принципы:
- DRY (Don’t Repeat Yourself)
- KISS (Keep It Simple, Stupid)
- YAGNI (You Aren’t Gonna Need It)

## Описание
В данном репозитории приведены примеры корректной реализации
принципов проектирования программного обеспечения на языке C#.

Каждый принцип реализован отдельно для наглядности и простоты понимания.
